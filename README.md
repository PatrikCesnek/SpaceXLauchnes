<h2>Hello there! I'm Patrik and this is my SpaceX launch demo app! 🚀</h2>

I chose programmatic UI instead of Storyboard, because I prefer to work with programatic UIs which gives me more control over how the app looks, 
I am proficient in Storyboards, this is simply a personal preference. 😊

Also I choose to make the detail in SwiftUI because for me it's easier to work with and more readable.
I prefer to use it whenever I can, yet I often suplement it with UIKit, especially in apps that need to support older devices. 

For projects like this, I prefer MVVM architecture for readability and ease of use.

I have the most experience with Codable, so that’s why I used it for decoding JSON data.

Most strings are placed in a Constants.swift file for safety. This makes it easier to update strings in one place if needed.
I ran into some minor issues I fixed along the way. Thinking about it I don't think there were any major issues I've had with this project. 
It was fun and down bellow you can see screenshots from the app. 😊
